# このリポジトリの目的
Bolt.jsを活用してSlackにChat GPTを活用したBotを作成する
基本的なアプリの構成は同じなのでこちらをベースにカスタマイズしたソースコードを各組織のリポジトリに追加する。

## はじめ方
```
npm i
npm run start
```

## 実行例
Chat-GPTのbotを呼び出し、スレッドに質問を行うと返答が返ってくる
